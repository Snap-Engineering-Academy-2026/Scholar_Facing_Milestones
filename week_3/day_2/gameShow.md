# GameShow API Project

Welcome to the **GameShow API Project**! Your goal is to build a **game show-style application** using **React** and **Vite** that interacts with a third-party API. Your app should allow users to control the API response using their input, and it should include a fun, game-like mechanic such as a score tracker, lives system, or leaderboard.

---

## Project Requirements

### 1. User Inputs

Include **at least two different types of user input** in your app:

- `Button`
- `Text Input`
- `Slider`
- `Checkbox`
- `Select Dropdown`

### 2. Dynamic API Calls

- Your app must use **at least one variable** to dynamically build the API request.
- You may need to manipulate:
  - **Endpoints** (e.g., `/api/city/London`)
  - or **Query Parameters** (e.g., `/api/search?difficulty=easy&category=books`)

### 3. Game Features

Include **at least one game-like mechanic**, such as:

- Score counter
- Lives system
- Timer
- Leaderboard

---

## Getting Started

### 1. Create Your Vite + React App

Open your terminal and run:

<details>
<summary> See Hints </summary>

```bash
# Create a new Vite app with React template
npm create vite@latest game-show-app -- --template react

# Move into your project directory
cd game-show-app

# Install dependencies
npm install

# Start the development server
npm run dev
```
</details>

### 2. Explore APIS

Explore free API's onlne find one that catches your interest. The API must be free to use.

### 3. Data Exploration

Look at the data you can get from the API (hint: use Postman) and come up with creative questions or interpretations. Come up with a few examples!
